开始

该文章介绍的是pc上富文本编辑的文档如何在微信小程序上使用，由于文章内容都是HTML DOM结构，小程序中没有相关的节点渲染，这样就需要HTML String转化成wxml可视化，wxParse实现了这个功能

注：如果需要转换的html节点较多，会造成短时间的空白，因为转化的处理过程需要点时间

使用方法

下载wxParse

打开https://github.com/icindy/wxParse通过git或者直接下载文件的方式下载到本地

使用方法

直接复制文件夹wxParse至自己项目里（可复制到components文件夹下）
注：下面的引入路径根据自己项目路径调节

原生小程序使用

在.wxss文件中引入样式

@import '../components/wxParse/wxParse.wxss';

在.js文件中引入模块

var WxParse = require('../components/wxParse/wxParse.js');

wepy使用方法

在.wpy文件中引入样式

@import '../components/wxParse/wxParse.wxss';

在.wpy文件中引入模块（全局或者当前函数）
